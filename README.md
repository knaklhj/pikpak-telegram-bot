## 一些实际使用中会碰到的问题

**问：磁力链接发给机器人没反应怎么办？**

先确认链接格式对不对（必须是 `magnet:?xt=urn:btih:` 开头）。如果格式没问题但机器人没响应，有可能是资源太冷门，PikPak服务器没有缓存，需要等一段时间或者换个资源试试。

**问：机器人提示"传输失败"是什么情况？**

可能原因有几个：资源本身已经失效（种子文件没有做种的人了），或者该资源被PikPak系统标记了（极少数情况）。大多数情况下，常规的BT/磁力资源成功率很高。

**问：免费用户可以用电报机器人吗？**

可以，机器人功能对免费用户也开放。区别在于下载速度和存储上限，如果你的6GB用完了，新的任务就存不进去了。

**问：移动宽带用户下载速度很慢怎么办？**

这是国内移动网络的老问题，移动运营商对PikPak域名有限速（据用户反映约30KB/s），联通相对好一些，电信也不太稳定。解决方法是走代理或者参考网上的"移动宽带下载限速破解"方案。
